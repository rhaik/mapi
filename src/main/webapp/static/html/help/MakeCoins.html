任务数量

每一天都有一定的APP不定时上线，且每一款APP的领取数量是有限的。

若您的网络不佳，有可能出现抢不到任务的情况，所以建议您多留意每一天推送的任务更新消息。

任务有限，好友无限

虽然每天的任务数量是有限的，但您的邀请好友数是没有限制的。所以在做任务的时候，记得多邀请好友一起来赚钱哦！

任务更新时间

每天会有一定量的应用不定时上线更新，未来会越来越多。商务正积极的拓展渠道，为大家尽力去争取更多的任务，希望您能理解。

怎样获取更多应用？

每款应用的投放量是有限的，可能会出现有的用户看不到试玩列表更新已经被抢光了。So，建议你平时多关注一下我们。

别人有，我没有？

在同一时间点，你朋友的任务列表有应用更新，而你没有应用更新。这种情况是因为你之前已经成功试玩过该应用，或在其它平台上已经下载过该应用。

任务有限，学徒无限

虽然每天的任务数量是有限的，但您的学徒数是没有限制的。所以在做任务的时候，记得多收徒弟一起来赚钱哦！
