操作步骤

1. 打开秒赚大钱app

2. 点击[限时任务]，选取当前可获取的任务，注意看任务描述，在列表中点击感兴趣的应用。

3. 跳转到应用详情后，按照提示认清提示的应用图标及位置，复制提示的关键词，再点击搜索下载系统会自动跳转到AppStore，找到该应用下载即可。

4. 打开应用试玩3分钟以上，等到系统提示你通过审核后再关闭应用，奖励现金到账。

试玩常见问题

1. 不可同时进行多个任务，同一时间只能进行单个任务，单个任务完成后方可开始下个任务。

2. 试玩用户过多时，到账提示会稍有延迟。

3. 平板电脑的用户下载应用试玩，一定要选择 “iPhone模式” 下载，否则会找不到应用任务或应用试玩审核不成功。

4. 如果你之前在手机上通过其它平台已经下载过该应用，在秒赚大钱看到后再下载，是不会有试玩成功提示和奖励的。
