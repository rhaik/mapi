正确的收徒方法

点击首页 – 收徒 打开页面，按照提示分享，选择发送给你的微信好友。

也可以点击收徒页面的图片版，将中意的图片保存后，可以在论坛、QQ等渠道邀请好友

师徒关系的建立

师徒关系的建立是用户通过你的邀请函或二维码首次关注或登录app。要确保在你邀请他之前，从未自己关注或通过其他人的邀请链接或二维码关注过。

你邀请的朋友关注或登录app后，即邀请成功。师徒关系可在收徒页面查看。

邀请收入说明

你的徒弟和你徒弟的徒弟（徒孙），他们所做的任务，你都可以得到最高20%的收益（由秒赚大钱官方补贴）。换言之，你的学徒做任务，不仅学徒全额收益，你也会得到官方额外给予的奖励！你可获得徒弟收益的20%官方额外补贴，可获得徒孙的收益的10%官方额外补贴。
