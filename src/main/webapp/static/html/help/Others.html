关于作弊

秒赚大钱禁止用户采用任何形式进行作弊，一经发现，我们将立即冻结作弊账号。如果您的操作一切正常但账号无法使用，请您联系客服申请账号解冻；如果您采取过任何不正当获取奖励的行为，您的账号将会被冻结，是无法正常使用的。

作弊等不正当行为包括但不限于：

1. 多设备操作：兑换时，一个支付宝只允许一个设备使用；

2. 使用共享付费ID：他人提供的已购买ID的信息获取奖励；

3. 越狱：越狱手机不支持；

4. 刷机：还原掉手机信息以重复获取奖励的行为。
