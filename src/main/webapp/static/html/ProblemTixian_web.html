常见问题
Q1:满多少钱提现？在哪提现？

A：满10元提现，秒赚大钱app里底部菜单栏【我】→【我的收入】→【提现】可查看

Q2:提现到哪？

A：目前支持提现到支付宝账户。

账户设置具体操作：

秒赚大钱app里底部菜单栏【我】→【设置】→【提现账号设置】→【支付宝】输入真实姓名和账号即可。

Q3:多久到账？

A：①提现一般2-4个工作日内到账。周末或节假日会顺延。

②由于打款进度不同，所以需要等待整批完成后，才会更新提现状态，所以可能会出现钱已到账，但是提现状态未更新的情况。

Q4:为何提现之后少了一元？

A：每笔提现支付宝会扣除1元手续费，所以你的到账收入比提现收入会少1元，请仔细确认。举例：如果你的提现金额是10元，则实际到账为9元。
